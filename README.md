# MySkills

个人维护的 Agent Skills 集合

## Skills

| Skill | Description |
|-------|------|
| [gpu-z](./gpu-z/) | 通过 GPU-Z 采集 GPU 硬件规格与实时传感器数据，诊断性能瓶颈 |

## 使用

将对应 skill 目录放到 Agent 的 skills 路径下即可，例如：

- Claude: `~/.claude/skills/<skill-name>/`
- Cursor: `~/.cursor/skills/<skill-name>/`

每个 skill 以 `SKILL.md` 为入口，详见各自目录内的 README。
